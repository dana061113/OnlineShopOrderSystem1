# OnlineShopOrderSystem1 (Java, SOLID)

Бұл репозиторийде интернет-дүкен үшін тапсырыстарды басқару жүйесінің қарапайым мысалы берілген.  
Жоба **SOLID принциптері** бойынша құрылған және Java тілінде жазылған.

## Жобаның мақсаты

- Тапсырыс (Order) құру
- Тауарларды қосу
- Әртүрлі төлем түрлерін қолдану (Credit Card, PayPal, Bank Transfer)
- Әртүрлі жеткізу тәсілдерін қолдану (Courier, Post, PickUp)
- Уведомление жіберу (Email, SMS)
- Скидканы есептеу (пайыздық және фиксированная)
- Барлығын **интерфейстер арқылы** кеңейтуге ыңғайлы етіп жасау
